# USER\_VECTOR\_LANG

The `USER_VECTOR_LANG` view displays all languages loaded by the current user.

|Column Name

|Data Type

|Description

|
|-------------|-----------|-------------|
|`LANG_LANG`

|`NUMBER`

|Language ID for the language \(for example, `1` for American\)

|
|`LANG_NAME`

|`VARCHAR2(128)`

|Name of the language \(for example, `AMERICAN`\)

|
