# VECTOR\_DIMENSION\_FORMAT

The `VECTOR_DIMENSION_FORMAT()` returns the storage format of the vector. It returns a `VARCHAR2`, which can be one of the following values: `'INT8'`, `'FLOAT32'`, or `'FLOAT64'`.
